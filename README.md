# Talabat Image Generator

A self-contained, browser-based toolkit for the GFX Team:

- **Prompt Builder** — auto-generates food photography prompts from item names
- **Batch Image Generator** — generates professional food images via FLUX.1
- **Logo Resizer** — 200×200px output
- **Cover Resizer** — 750×300px output
- **Batch Resizer** — process up to 500 images at once with Auto Smart Trim
- **Background Remover** — runs fully client-side, no upload needed

No backend required. Everything runs in the browser.
