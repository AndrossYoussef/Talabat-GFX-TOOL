# 🚀 Talabat GFX Tool — GitHub Pages Deployment

## Why GitHub Pages?
- ✅ Free forever, no expiry
- ✅ GitHub is rarely blocked (unlike Netlify)
- ✅ HTTPS automatic
- ✅ Custom domain support later if needed

---

## Step 1 — Create GitHub account
1. Go to: https://github.com/signup
2. Sign up with your email (personal Gmail is fine, not company email)
3. Choose a username (e.g. `andro-talabat` or similar)

---

## Step 2 — Create a new repository
1. Click the **"+"** icon top right → **"New repository"**
2. Repository name: `talabat-gfx-tool`
3. Keep it **Public** (required for free GitHub Pages)
4. Click **"Create repository"**

---

## Step 3 — Upload the files
1. On the new repo page, click **"uploading an existing file"**
2. Drag and drop:
   - `index.html`
   - `README.md`
3. Scroll down → click **"Commit changes"**

---

## Step 4 — Enable GitHub Pages
1. In your repo, click **"Settings"** (top menu)
2. Click **"Pages"** (left sidebar)
3. Under **"Branch"** → select **"main"** → folder **"/ (root)"**
4. Click **"Save"**

---

## Step 5 — Wait ~1 minute, then visit your link!

Your tool will be live at:
```
https://[your-username].github.io/talabat-gfx-tool/
```

Example: `https://andro-talabat.github.io/talabat-gfx-tool/`

---

## 🔄 To update the tool later
1. Go to your repo → click `index.html`
2. Click the pencil ✏️ icon (Edit)
3. Paste the new code → Commit changes
4. Wait ~1 minute, refresh the page
